#### Getting Started

* Before Start Project

```ssh
$ git clone ....`
$ cd ts-react-library-skeleton
$ npm install
```
* run start Showcase (Development Mode) 
```ssh
$ npm start
```

* run test (Test Mode) 

```ssh
$ npm test
```

* generate Showcase build (Production Mode) 
```ssh
$ npm start
```

* run coverage

```ssh
$ npm run coverage
```

* generate api docs

```ssh
$ npm run docs
```

* build library as javascript (common-js)

```ssh
$ npm build
```
