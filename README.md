#### Getting Started

* Before Start Project

```ssh
$ git clone ....`
$ cd ts-react-library-skeleton
$ npm install
```

* run test

```ssh
$ npm test
```

* run coverage

```ssh
$ npm run coverage
```


* export docs

```ssh
$ npm run docs
```

* build code as javascript (common-js)

```ssh
$ npm build
```
